CH 02 · 네이버 검색광고

검색하는 사람에게
정확히 노출되기

메타가 관심 기반이라면, 네이버는 검색 기반입니다. "내 서비스를 찾고 있는 사람"에게 보여주는 광고입니다.

INTRO
네이버 검색광고 구조
네이버도 메타와 같은 3단계 구조입니다.
Level 1
캠페인
광고 목적 단위
(사이트 검색광고 /
쇼핑검색 / 브랜드)
Level 2
광고그룹
키워드·입찰가·타겟
"어떤 키워드 묶음"
Level 3
키워드·소재
실제 검색어 + 광고 문구
검색 시 노출되는 광고
💡
메타 vs 네이버 선택 기준: 내 서비스를 "검색하는" 사람이 있다면 네이버, 아직 필요성을 모르는 사람에게 알려야 한다면 메타. 둘 다 병행하면 더 좋습니다.
STEP 1
네이버 광고 계정 만들기
searchad.naver.com에서 진행합니다. 네이버 계정이 있으면 바로 시작 가능합니다.
1
네이버 광고 시스템 접속
searchad.naver.com 접속 → 네이버 계정으로 로그인 → "광고 시작하기".
searchad.naver.com
2
비즈채널 등록
광고할 사이트 URL을 등록합니다. 네이버가 사이트를 검수합니다 (보통 1~3일 소요). 검수 완료 후 광고 집행 가능.
광고 관리 → 비즈채널 관리 → 채널 추가
3
충전금 입금
네이버 광고는 선불제입니다. 광고비를 먼저 충전해야 집행됩니다. 최소 충전금: 10만원. 처음엔 10~20만원으로 시작 권장.
⚠️
비즈채널 검수는 1~3일 걸립니다. 광고를 바로 시작하고 싶다면 미리 등록해두세요. 검수 중 사이트 내용을 바꾸면 재검수가 필요합니다.
STEP 2
캠페인 → 광고그룹 만들기
서비스 초기에는 "사이트 검색광고" 유형 하나만 시작하세요.
1
캠페인 만들기
광고 관리 → 캠페인 → "새 캠페인 만들기" → 유형: 파워링크(사이트 검색광고) 선택. 이름: 브랜드명_서비스명 형식으로.
광고 관리 → 캠페인 → 새 캠페인 만들기
2
일 예산 설정
하루에 최대 얼마까지 쓸지 설정합니다. 처음엔 3~5만원으로 시작. 이 금액이 소진되면 그날 광고는 자동으로 중지됩니다.
3
광고그룹 만들기
캠페인 안에 광고그룹을 만듭니다. 서비스 카테고리별로 그룹을 나누세요. 예: "브랜드 키워드 그룹", "일반 키워드 그룹".
STEP 3
키워드 선택하기
검색광고의 핵심입니다. 어떤 단어를 검색했을 때 내 광고가 뜨게 할지 결정합니다.
브랜드 키워드
내 브랜드명을 검색하는 사람에게 노출. 경쟁사가 내 브랜드명으로 광고하는 걸 막는 효과도 있음. 반드시 집행.
파인디어랩
서비스 키워드
내 서비스를 직접 찾는 사람. 전환 의도가 높아 비용 대비 효율이 좋음.
브랜드 컨설팅
문제 키워드
고객이 가진 문제를 검색하는 사람. 경쟁은 낮지만 전환까지 설득이 필요.
스타트업 브랜딩
비교 키워드
다른 서비스와 비교하거나 후기를 찾는 사람. 의사결정 단계에 있는 고객.
브랜드 대행사 비교
💡
키워드 플래너 활용: searchad.naver.com → 도구 → 키워드 도구. 키워드 검색량과 예상 입찰가를 미리 확인할 수 있습니다.
⚠️
키워드는 적게, 연관도 높게. 처음에 100개 키워드를 넣는 실수를 많이 합니다. 연관성 낮은 키워드는 낭비입니다. 10~20개의 핵심 키워드로 시작하세요.
STEP 4
광고 소재(문구) 작성하기
제목 2줄 + 설명 1줄로 구성됩니다. 글자 수 제한이 있습니다.
제목 1 (15자 이내)
키워드 포함 + 핵심 혜택
브랜드 전략 컨설팅 — 키워드를 그대로 포함하면 볼드 처리되어 눈에 띕니다.
제목 2 (15자 이내)
차별점 또는 행동 유도
사장님이 직접 운영하는 구조 — 경쟁사와 다른 이유를 한 줄로.
설명 (45자 이내)
구체적 내용 + 클릭 이유 제공
로고 전에 브랜드 구조부터 설계합니다. 대행 말고, 직접 운영하는 브랜드를 만드세요.
💡
소재는 2~3개 만들어 테스트하세요. 네이버가 성과 좋은 소재를 자동으로 더 자주 노출합니다.
STEP 5
입찰가 설정하기
입찰가는 클릭 한 번에 지불할 최대 금액입니다. 높을수록 상단에 노출됩니다.
1
키워드별 입찰가 확인
키워드 도구에서 각 키워드의 "예상 월간 검색수"와 "평균 클릭 비용"을 확인합니다. 경쟁이 높은 키워드는 클릭당 3,000~10,000원까지 나옵니다.
2
브랜드 키워드 입찰가
내 브랜드 키워드는 경쟁이 낮아 100~300원 수준. 반드시 1~3위 유지. 낮게 잡으면 경쟁사가 내 브랜드명 광고에서 상단 차지 가능.
3
자동 입찰 (추천)
처음에는 "스마트 입찰" 사용 권장. 설정한 목표에 맞게 네이버가 자동으로 입찰가를 조정합니다.
광고그룹 설정 → 입찰가 전략 → 스마트 입찰
STEP 6
성과 확인하기
네이버에서 꼭 봐야 할 지표입니다.
→
노출수 / 클릭수 / CTR
몇 명에게 보여졌고 몇 명이 클릭했는지. CTR(클릭률) 업종 평균은 1~3%. 이 수치가 낮으면 소재 문구를 수정해야 합니다.
→
평균 노출 순위
내 광고가 검색 결과 몇 번째에 뜨는지. 1~3위 유지가 목표. 4위 이하면 클릭이 급격히 감소합니다.
→
전환 추적 설정
광고 관리자 → 도구 → 전환 추적. 문의 페이지 방문이나 전화 클릭 등을 "전환"으로 설정하면 어떤 키워드가 실제 문의를 만드는지 알 수 있습니다.
CHECK
광고 집행 전 체크리스트
네이버 광고 계정 생성 완료
비즈채널 검수 완료 (1~3일)
충전금 입금 완료 (최소 10만원)
핵심 키워드 10~20개 선정 완료
광고 소재(제목+설명) 2종 작성 완료
일 예산 설정 완료

CH 03 · GA4 데이터 읽기

숫자가 말하는
것을 읽는 법

광고를 돌리면 데이터가 쌓입니다. 어떤 숫자를 봐야 하는지, 그 숫자가 무엇을 의미하는지 알아야 합니다.

METRICS
반드시 알아야 할 5가지 지표
GA4 보고서에서 이 5개만 먼저 이해하세요.
사용자
Users (순 방문자)
내 사이트에 방문한 실제 사람 수. 같은 사람이 3번 방문해도 1명으로 계산됩니다.
핵심 지표 — 매일 확인
세션
Sessions (방문 횟수)
사이트 방문 횟수. 1명이 오늘 3번 방문하면 세션 3. 사용자보다 세션이 크면 재방문이 있다는 뜻.
참여율
Engagement Rate
10초 이상 머물거나 2페이지 이상 본 세션 비율. 이 수치가 높을수록 콘텐츠가 좋다는 신호. 50% 이상 목표.
목표: 50% 이상
평균 참여 시간
Avg. Engagement Time
방문자가 사이트에서 실제로 활동한 시간. 1분 이상이면 콘텐츠를 읽고 있다는 의미.
목표: 1분 이상
트래픽 소스
Session Source / Medium
방문자가 어디서 왔는지. Organic Search(검색), Direct(직접), Referral(링크), Paid Social(광고) 등으로 구분. 광고 집행 후 Paid Social 유입이 늘어야 합니다.
광고 효과 확인 필수
REPORTS
자주 쓰는 보고서 경로
GA4 왼쪽 메뉴 → 보고서에서 찾아가는 경로입니다.
실시간 현황
보고서 → 실시간 — 지금 이 순간 몇 명이 접속 중인지 확인
전체 방문자
보고서 → 수명 주기 → 획득 → 개요 — 기간별 방문자 수
유입 채널
보고서 → 수명 주기 → 획득 → 트래픽 획득 — 어디서 왔는지
페이지별 성과
보고서 → 수명 주기 → 참여 → 페이지 및 화면 — 어느 페이지를 많이 봤는지
방문자 위치
보고서 → 사용자 → 인구통계 → 인구통계 개요 — 어느 지역에서 왔는지
ADV CHECK
광고 효과를 GA4에서 확인하는 법
광고를 집행하면 GA4에서 어느 광고에서 방문했는지 추적됩니다.
1
트래픽 획득 보고서 확인
보고서 → 획득 → 트래픽 획득 보고서에서 "Paid Social" 또는 "Paid Search" 행을 확인합니다. 메타 광고는 Paid Social, 네이버는 Paid Search로 잡힙니다.
보고서 → 획득 → 트래픽 획득
2
UTM 파라미터로 광고 구분하기
여러 광고를 동시에 돌릴 때 어느 광고에서 왔는지 구분하려면 UTM을 사용합니다. 광고 URL 뒤에 ?utm_source=instagram&utm_medium=paid&utm_campaign=브랜딩 처럼 붙입니다.
3
광고 유입과 일반 유입 비교
광고로 온 사람과 검색으로 온 사람의 참여율, 체류 시간을 비교해보세요. 광고 유입의 참여율이 현저히 낮다면 랜딩 페이지 개선이 필요합니다.
💡
광고비 대비 방문자 계산: GA4 광고 유입 수 ÷ 광고비 = 방문자 1명당 비용(CPC). 이 수치를 네이버 광고 관리자의 CPC와 비교하면 집계 차이를 발견할 수 있습니다.
ALERT
이상 신호 감지하기
이런 상황이 보이면 빠르게 대응해야 합니다.
!
방문자 갑자기 0명
GA4 추적 코드가 삭제됐거나 사이트가 다운된 경우. 즉시 사이트 접속 확인 → GA4 코드 설치 확인.
!
참여율 10% 미만
방문자는 오지만 바로 나가는 상황. 광고 소재와 랜딩 페이지 내용이 다를 때 발생. 광고 문구와 페이지 내용을 일치시키세요.
!
광고 유입이 GA4에 안 잡힘
메타 광고 → 메타 광고 관리자에는 클릭 수가 있는데 GA4 유입이 없다면 랜딩 URL에 GA4가 설치 안 된 페이지일 수 있습니다. 랜딩 페이지 코드 확인 필요.

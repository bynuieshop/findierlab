광고 전에
반드시 해야 할 것

광고를 돌리기 전, 측정 도구부터 설치해야 합니다. GA4와 메타 픽셀이 없으면 광고비가 어디로 갔는지 알 수 없습니다.

WHY
측정 없이 광고하면 안 되는 이유
광고를 켜기 전 5분을 투자하면, 광고비 낭비를 막을 수 있습니다.
→
GA4 (구글 애널리틱스)
내 사이트에 누가 왔는지, 어느 페이지를 봤는지, 얼마나 머물렀는지 기록합니다. 광고 채널 구분 없이 전체 방문자 데이터를 한 곳에서 봅니다.
→
메타 픽셀
인스타그램·페이스북 광고를 클릭한 사람이 내 사이트에서 어떤 행동을 했는지 메타에 전달합니다. 픽셀이 없으면 메타가 광고를 최적화할 데이터가 없어 성과가 낮아집니다.
💡
이미 설치했다면 넘어가세요. 이전에 개발자나 대행사가 설치해 뒀을 수 있습니다. 설치 여부 확인은 각 섹션 끝에 안내합니다.
STEP 1
GA4 계정 만들기
구글 계정이 있으면 무료로 만들 수 있습니다. 약 10분 소요.
1
Google Analytics 접속
analytics.google.com 접속 후 구글 계정으로 로그인합니다.
analytics.google.com
2
계정 만들기 클릭
"측정 시작" 또는 "계정 만들기" 버튼을 클릭합니다.
측정 시작
3
계정 이름 입력
계정 이름: 브랜드명 또는 회사명 (예: 파인디어랩). 계정은 여러 사이트를 묶는 최상위 단위입니다.
4
속성 만들기
속성 이름: 사이트 주소 (예: findierlab.com). 시간대: 한국, 통화: KRW(원) 선택.
5
플랫폼 선택
"웹"을 선택합니다. 앱이 없는 경우 웹만 선택하면 됩니다.
6
측정 ID 확인
완료하면 G-XXXXXXXXXX 형태의 측정 ID가 발급됩니다. 이것이 내 GA4 고유 코드입니다.
STEP 2
GA4를 사이트에 설치하기
발급받은 코드를 사이트에 넣어야 방문자가 기록되기 시작합니다.
1
직접 코드 삽입 (HTML 사이트)
사이트가 HTML 파일로 되어 있다면, 모든 페이지의 <head> 태그 안에 아래 코드를 추가합니다.
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){"{"}dataLayer.push(arguments);{"}"}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXX');
</script>
2
워드프레스 사용 시
플러그인 "Site Kit by Google" 설치 → 구글 계정 연결 → 자동 설치됩니다. 코드를 직접 건드릴 필요 없습니다.
플러그인 → Site Kit by Google → 활성화
3
설치 확인하기
GA4 → 관리 → 데이터 스트림 → 웹스트림 선택 → 태그 안내 옆 "태그 작동 여부 테스트" 클릭. 초록 체크 뜨면 정상입니다.
⚠️
설치 후 24시간은 기다려야 합니다. GA4는 실시간 보고서에서 먼저 확인되고, 다음 날부터 정식 데이터가 보입니다. 설치 직후 빈 화면이어도 정상입니다.
STEP 3
메타 픽셀 만들기
메타 광고를 돌리기 전 반드시 필요합니다. 약 10분 소요.
1
Meta Business Suite 접속
business.facebook.com 접속 후 페이스북 계정으로 로그인합니다. 처음이라면 비즈니스 계정을 먼저 만들어야 합니다.
business.facebook.com
2
이벤트 관리자 이동
좌측 메뉴 → 이벤트 관리자 (Events Manager) 클릭.
좌측 메뉴 → 이벤트 관리자
3
데이터 소스 연결
"데이터 소스 연결" 또는 "+ 추가" 클릭 → "웹" 선택 → 픽셀 이름 입력 (예: 파인디어랩) → 사이트 URL 입력.
4
픽셀 ID 확인
생성 완료 후 15자리 숫자 Pixel ID가 발급됩니다. 이 번호를 복사해 두세요.
STEP 4
메타 픽셀을 사이트에 설치하기
발급받은 픽셀 코드를 사이트 모든 페이지에 삽입합니다.
1
HTML 사이트에 직접 삽입
모든 페이지의 </head> 태그 바로 앞에 아래 코드를 추가합니다. PIXEL_ID 부분을 발급받은 숫자로 교체하세요.
<script>
  !function(f,b,e,v,n,t,s){"{"}...{"}"}(window, document,'script',
  'https://connect.facebook.net/en_US/fbevents.js');
  fbq('init', 'PIXEL_ID');
  fbq('track', 'PageView');
</script>
2
픽셀 작동 확인
Chrome 확장 프로그램 "Meta Pixel Helper" 설치 후 내 사이트 접속. 초록 아이콘 뜨면 정상 작동입니다.
💡
코드 설치가 어려우면: 이벤트 관리자에서 "파트너 통합" 선택 → Squarespace, Wix, Shopify 등 플랫폼 자동 연결 옵션이 있습니다.
CHECK
다음 단계 전 체크리스트
아래 항목이 모두 완료되면 메타 광고를 시작할 준비가 됐습니다.
GA4 계정 생성 완료
GA4 측정 ID (G-XXXXXXXX) 발급됨
GA4 코드가 사이트 모든 페이지에 삽입됨
메타 픽셀 생성 완료
픽셀 ID (15자리 숫자) 발급됨
메타 픽셀 코드가 사이트 모든 페이지에 삽입됨
